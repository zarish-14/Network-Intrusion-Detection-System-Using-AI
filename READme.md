## Project Description
This project implements different Artificial Intelligence and Machine Learning techniques for Network
Intrusion Detection using the dataset network_traffic.csv.
The following tasks are included:
## Task 1: Data Exploration
## Task 2: Simple Reflex Agent
Task 3A: K-Nearest Neighbors (KNN)
Task 3B: Naive Bayes
Task 3C: Logistic Regression
Task 4: K-Means Clustering
## Task 5: Genetic Algorithm
## Requirements
Install the following Python libraries before running the project:
pipinstallpandasnumpymatplotlib
## Dataset
Place the dataset file:
network_traffic.csv
in the same folder as the Jupyter Notebook.

How to Run
## Step 1
## Open Jupyter Notebook:
jupyternotebook
## Step 2
Open the project notebook.
## Step 3
Run all cells from top to bottom using:
## Kernel → Restart & Run All
or run each task cell individually.

Dataset statistics
